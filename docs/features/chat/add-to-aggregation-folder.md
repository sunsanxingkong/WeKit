# 添加对话至归拢文件夹

> 在首页对话列表长按菜单添加菜单项, 可将该对话加入「对话归拢」的手动文件夹
> 需启用「对话归拢」

## 类别

聊天

## 类型

可开关 & 可点击

## 描述

在首页对话列表的长按菜单中添加「加入文件夹」菜单项。选中对话后, 会将指定对话加入「对话归拢」功能创建的手动文件夹中。

可选设置「添加后打开配置对话框」, 在加入文件夹后自动弹出该文件夹的编辑对话框, 方便调整文件夹设置。

## 使用方法

在模块设置中启用, 确保同时启用「对话归拢」。在首页长按任意对话, 菜单中选择「加入文件夹」, 然后选择目标文件夹。可点击「添加对话至归拢文件夹」功能以配置「添加后打开配置对话框」选项
