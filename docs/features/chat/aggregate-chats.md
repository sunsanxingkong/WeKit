# 对话归拢

> 将多个对话归拢在一个文件夹内
> 设置对话头像需同时启用「自定义好友本地头像」

## 类别

聊天

## 类型

可开关 & 可点击

## 描述

在主页对话列表上方创建可自定义的文件夹, 将多个对话归拢到一起, 实现类似聊天软件「文件夹」或「分类」的功能。

每个文件夹可设置名称, 并包含指定的对话列表。文件夹在首页对话列表中作为一个独立条目显示, 并带有未读角标。文件夹内对话的未读计数会聚合到文件夹条目上。

文件夹中的对话支持:
- 置顶 / 移动
- 消息实时更新 (有新消息时文件夹条目自动刷新)
- 从文件夹中移除 (文件夹右键菜单)
- 自定义头像 (需同时启用「自定义好友本地头像」— 文件夹头像默认使用其中一个成员的头像)

## 使用方法

在模块设置中启用后, 点击「对话归拢」打开管理对话框, 可创建、编辑和删除文件夹。也可在首页对话列表长按菜单通过「加入文件夹」将对话加入已有文件夹
