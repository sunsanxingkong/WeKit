# 分享进化

> 让微信的系统分享菜单更易用 (没写完)

## 类别

聊天

## 类型

可开关 & 可点击

## 描述

通过 Android Dynamic Shortcuts 机制, 将聊天最频繁的联系人注入到系统分享菜单的「直接分享」目标中。在其他应用中点击「分享到微信」时, 系统分享菜单会直接显示这些联系人的快捷入口, 无需经过联系人选择界面即可一键分享。

系统会自动根据消息互动量选取排名前 3 的好友生成快捷方式。

点击功能可清除所有已注册的动态快捷方式。

⚠️ **该功能尚未完成**。

## 使用方法

在模块设置中启用。重新打开系统分享菜单即可看到常用联系人快捷入口
