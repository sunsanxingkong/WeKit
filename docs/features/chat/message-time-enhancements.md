# 消息时间增强

> 显示精确消息发送时间并允许显示更多详情

## 类别

聊天

## 类型

可开关 & 可点击

## 描述

增强聊天消息中时间戳的显示方式。默认微信只显示「昨天」「10:30」等模糊时间, 本功能可配置为显示完整日期和精确时间, 还可自定义显示更多消息详情信息。

可配置项:
- **时间格式**: 自定义时间显示格式 (如 `yyyy/MM/dd HH:mm:ss`)
- **显示格式**: 自定义格式模板, 支持 `$time` (时间)、`$type` (消息类型)、`$id` (消息 ID)、`$sender` (发送者) 等变量
- **文字大小**: 调整时间文字的大小
- **文字颜色**: 浅色/深色模式下分别设置
- **始终居中**: 时间显示位置始终居中

## 使用方法

在模块设置中启用后, 点击配置时间格式和显示内容
