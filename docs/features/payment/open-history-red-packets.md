# 捡漏历史红包

> 在联系人与群组详情页面添加选项, 批量扫描当前对话的所有历史红包消息并尝试领取
> 点击可查看当前正在进行的任务

## 类别

红包与支付

## 类型

可开关 & 可点击

## 描述

在联系人与群组的详情页面添加「捡漏历史红包」入口, 可以扫描当前对话中所有历史红包消息（包括已领完和已过期的）, 并尝试逐个领取。适用于错过群红包后回来捡漏的场景。

功能特性:
- 自动扫描本地聊天记录中的红包消息
- 逐个发起领取请求
- 实时显示扫描和领取进度
- 可随时取消正在进行的任务
- 点击功能图标可查看当前任务状态

## 使用方法

在模块设置中启用。在任意好友或群聊的详情页中, 点击「捡漏历史红包」开始扫描。点击模块设置中的「捡漏历史红包」可查看当前正在进行的任务进度
