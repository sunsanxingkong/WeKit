# 转发 & 一键转发

> 转发他人的朋友圈
> 如果图片/视频转发后是空白, 请点击查看/播放后重试

## 类别

朋友圈

## 类型

可开关

## 描述

在朋友圈的长按菜单中添加「转发」和「一键转发」选项, 可以将好友的朋友圈内容转发到自己的朋友圈。

- **转发**: 打开朋友圈编辑界面, 将内容填入后由你手动发布
- **一键转发**: 自动以相同内容快速发布, 无需手动编辑

支持转发文字、图片和视频内容。如果图片或视频转发后显示空白, 请先点击原动态的图片/视频查看或播放后再重试。

## 使用方法

在模块设置中启用。在朋友圈长按任意动态, 选择「转发」或「一键转发」
