# 自定义消息气泡

> 自定义聊天中的消息气泡图片和颜色

## 类别

界面美化, 聊天

## 类型

可开关 & 可点击

## 描述

自定义聊天消息气泡的外观。支持为「自己的消息」和「对方的消息」分别设置:

- **文字颜色** — 浅色/深色模式下独立设置
- **气泡背景色** — 可设置纯色背景
- **气泡图片** — 使用自定义 `.9.png` 图片替换气泡背景

支持的图片消息类型包括: 文本、链接、群通知、引用消息、音视频通话、语音消息。将图片文件放入 `<模块数据>/assets/` 目录, 命名为 `left_bubble.9.png` (对方气泡) 或 `right_bubble.9.png` (自己气泡) 即可生效。

## 使用方法

在模块设置中启用后, 点击配置颜色和气泡图片
